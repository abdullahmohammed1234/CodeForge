Numbers

In Python, numbers are fundamental data types used for mathematical operations. Python supports several numeric types, with integers and floating-point numbers being the most common.

Numeric Types in Python

  • int: Integer numbers (positive, negative, or zero)
  • float: Floating-point numbers (decimal numbers)
  • complex: Complex numbers (with real and imaginary parts)

Integers (int)

Integers are whole numbers without decimal points:

  • Can be positive, negative, or zero
  • No size limit in Python 3 (limited only by available memory)
  • Can be written in different bases (decimal, binary, octal, hexadecimal)
# Integer examples
positive_int = 42
negative_int = -17
zero = 0

# Different bases
decimal = 255
binary = 0b11111111  # 255 in binary
octal = 0o377        # 255 in octal
hexadecimal = 0xff   # 255 in hexadecimal

print(decimal, binary, octal, hexadecimal)  # All output 255

Floating-Point Numbers (float)

Floats represent decimal numbers:

  • Based on IEEE 754 double-precision standard
  • Can represent very large and very small numbers
  • May have precision issues due to binary representation
  • Can be written in scientific notation
# Float examples
pi = 3.14159
negative_float = -2.5
scientific = 1.23e4  # 12300.0
small_number = 1.23e-4  # 0.000123

print(pi, negative_float, scientific, small_number)

Complex Numbers (complex)

Complex numbers have real and imaginary parts:

# Complex number examples
complex_num = 3 + 4j
real_part = complex_num.real  # 3.0
imag_part = complex_num.imag  # 4.0

print(complex_num)  # (3+4j)
print(real_part, imag_part)

Type Conversion

You can convert between numeric types:

# Type conversion
int_to_float = float(42)    # 42.0
float_to_int = int(3.14)    # 3 (truncates decimal part)
int_to_complex = complex(5) # (5+0j)

# String to number
str_to_int = int("123")
str_to_float = float("3.14")

print(int_to_float, float_to_int, int_to_complex)

Arithmetic Operations

Basic arithmetic operations work with all numeric types:

a = 10
b = 3

print("Addition:", a + b)        # 13
print("Subtraction:", a - b)     # 7
print("Multiplication:", a * b)  # 30
print("Division:", a / b)        # 3.333...
print("Floor Division:", a // b) # 3
print("Modulus:", a % b)         # 1
print("Exponentiation:", a ** b) # 1000

Built-in Functions

Python provides useful built-in functions for numbers:

numbers = [3.14, -2.5, 10, 7.8]

print("Absolute value:", abs(-5))     # 5
print("Round:", round(3.14159, 2))    # 3.14
print("Power:", pow(2, 3))            # 8
print("Maximum:", max(numbers))       # 10
print("Minimum:", min(numbers))       # -2.5
print("Sum:", sum(numbers))           # 18.44

Math Module

The math module provides advanced mathematical functions:

import math

print("Square root:", math.sqrt(16))     # 4.0
print("Sine:", math.sin(math.pi/2))      # 1.0
print("Cosine:", math.cos(0))            # 1.0
print("Logarithm:", math.log(100, 10))   # 2.0
print("Factorial:", math.factorial(5))   # 120
print("Ceiling:", math.ceil(3.2))        # 4
print("Floor:", math.floor(3.8))         # 3

Precision and Limitations

  • Floats have limited precision (about 15 decimal digits)
  • Some decimal numbers can't be represented exactly in binary
  • Use decimal module for financial calculations requiring exact decimal representation
# Precision example
print(0.1 + 0.2)  # 0.30000000000000004 (not exactly 0.3)

# Using decimal for precision
from decimal import Decimal
a = Decimal('0.1')
b = Decimal('0.2')
print(a + b)  # 0.3 (exact)

Mini Project

Create a number analyzer program:

  • Ask the user to input a number
  • Determine if it's positive, negative, or zero
  • Check if it's an integer or float
  • Calculate square, square root, and cube
  • Display the number in binary, octal, and hexadecimal
  • Bonus: Handle complex numbers

Here's a starting template:

import math

# Get user input
num_str = input("Enter a number: ")

try:
    num = float(num_str)
    
    # Check type and sign
    if num > 0:
        sign = "positive"
    elif num < 0:
        sign = "negative"
    else:
        sign = "zero"
    
    print(f"The number {num} is {sign}")
    
    # Calculations
    if num >= 0:
        print(f"Square root: {math.sqrt(num)}")
    
    print(f"Square: {num ** 2}")
    print(f"Cube: {num ** 3}")
    
    # Different bases (for integers)
    if num == int(num):
        int_num = int(num)
        print(f"Binary: {bin(int_num)}")
        print(f"Octal: {oct(int_num)}")
        print(f"Hexadecimal: {hex(int_num)}")

except ValueError:
    print("Invalid number entered")
