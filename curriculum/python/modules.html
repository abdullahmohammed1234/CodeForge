Modules

Modules in Python are files containing Python code that can be imported and used in other Python programs. They allow you to organize code into reusable components, making programs more modular and maintainable.

What are Modules?

  • Files with .py extension containing Python code
  • Can define functions, classes, and variables
  • Can be imported into other Python programs
  • Promote code reusability and organization
  • Help avoid naming conflicts

Creating Modules

Any Python file can be used as a module. Here's an example:

# mymodule.py
def greet(name):
    return f"Hello, {name}!"

def add_numbers(a, b):
    return a + b

PI = 3.14159

class Calculator:
    def multiply(self, a, b):
        return a * b

Importing Modules

There are several ways to import modules:

# Method 1: Import the entire module
import mymodule

print(mymodule.greet("Alice"))  # Hello, Alice!
print(mymodule.add_numbers(5, 3))  # 8
print(mymodule.PI)  # 3.14159

calc = mymodule.Calculator()
print(calc.multiply(4, 5))  # 20

# Method 2: Import specific items
from mymodule import greet, add_numbers

print(greet("Bob"))  # Hello, Bob!
print(add_numbers(10, 7))  # 17

# Method 3: Import with alias
import mymodule as mm

print(mm.greet("Charlie"))  # Hello, Charlie!

# Method 4: Import all (not recommended)
from mymodule import *

print(greet("David"))  # Hello, David!

The Standard Library

Python comes with a rich standard library of modules:

  • math: Mathematical functions
  • datetime: Date and time handling
  • os: Operating system interface
  • sys: System-specific parameters
  • random: Random number generation
  • json: JSON encoding/decoding
  • re: Regular expressions
import math
import datetime
import os
import random

print("Square root:", math.sqrt(16))
print("Current date:", datetime.date.today())
print("Current directory:", os.getcwd())
print("Random number:", random.randint(1, 10))

Module Search Path

Python searches for modules in the following order:

  1. Current directory
  2. PYTHONPATH environment variable
  3. Python installation directory
  4. Site-packages directory
import sys

print("Module search paths:")
for path in sys.path:
    print(path)

Creating Packages

Packages are directories containing multiple modules:

# Directory structure:
# mypackage/
#   __init__.py
#   module1.py
#   module2.py
#   submodule/
#       __init__.py
#       module3.py

# __init__.py (can be empty or contain initialization code)
# module1.py
def function1():
    return "Function 1"

# Usage
import mypackage.module1
print(mypackage.module1.function1())

# Or with __init__.py defining imports
from mypackage import module1
print(module1.function1())

__name__ and __main__

The __name__ variable helps determine how a module is being used:

# mymodule.py
def main():
    print("This is the main function")

if __name__ == "__main__":
    main()
    print("Module run directly")
else:
    print("Module imported")

# When run directly: python mymodule.py
# Output: This is the main function
#         Module run directly

# When imported: import mymodule
# Output: Module imported

Installing Third-Party Modules

Use pip to install external packages:

# Install a package
pip install requests

# Install specific version
pip install requests==2.25.1

# Install from requirements file
pip install -r requirements.txt

# List installed packages
pip list

# Show package information
pip show requests

Virtual Environments

Virtual environments isolate project dependencies:

# Create virtual environment
python -m venv myenv

# Activate (Windows)
myenv\Scripts\activate

# Activate (macOS/Linux)
source myenv/bin/activate

# Install packages in virtual environment
pip install requests

# Deactivate
deactivate

Mini Project

Create a custom module for text processing:

  • Create a module called text_utils.py
  • Implement functions for counting words, characters, and sentences
  • Add functions to convert case and remove punctuation
  • Create a main function to demonstrate usage
  • Import and use the module in another script
  • Bonus: Add support for reading from files

Here's a starting template:

# text_utils.py
import string

def count_words(text):
    """Count the number of words in text."""
    words = text.split()
    return len(words)

def count_characters(text):
    """Count total characters in text."""
    return len(text)

def count_sentences(text):
    """Count the number of sentences in text."""
    sentences = text.split('.')
    return len([s for s in sentences if s.strip()])

def to_uppercase(text):
    """Convert text to uppercase."""
    return text.upper()

def to_lowercase(text):
    """Convert text to lowercase."""
    return text.lower()

def remove_punctuation(text):
    """Remove punctuation from text."""
    return text.translate(str.maketrans('', '', string.punctuation))

def main():
    """Demonstrate the text utilities."""
    sample_text = "Hello, World! This is a sample text. It has multiple sentences."
    
    print("Original text:", sample_text)
    print("Word count:", count_words(sample_text))
    print("Character count:", count_characters(sample_text))
    print("Sentence count:", count_sentences(sample_text))
    print("Uppercase:", to_uppercase(sample_text))
    print("No punctuation:", remove_punctuation(sample_text))

if __name__ == "__main__":
    main()

# Usage in another file:
# import text_utils
# text_utils.main()
