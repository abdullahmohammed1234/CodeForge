If Else Statements

If-else statements are fundamental to programming logic, allowing your code to make decisions and execute different blocks of code based on conditions. In Python, these conditional statements use boolean expressions to determine which path your program should take. Mastering if-else statements is crucial for creating dynamic and responsive programs.

What are Conditional Statements?

Conditional statements allow your program to:

  • Execute code only when certain conditions are met
  • Choose between different actions based on input or data
  • Handle different scenarios gracefully
  • Create interactive programs that respond to user input
  • Implement business logic and decision-making processes

The if Statement

The basic if statement executes code only if a condition is true:

age = 18

if age >= 18:
    print("You are an adult!")
    print("You can vote and drive.")

print("This always executes")

Key Points:

  • The condition must evaluate to True or False
  • Code inside the if block must be indented
  • Python uses colon (:) after the condition
  • Statements after the if block execute regardless

The if-else Statement

The if-else statement provides two paths of execution:

temperature = 25

if temperature > 30:
    print("It's hot outside!")
    print("Stay hydrated.")
else:
    print("The weather is pleasant.")
    print("Enjoy your day!")

print("Weather check complete.")

The elif Statement

elif (else if) allows multiple conditions to be checked:

score = 85

if score >= 90:
    grade = "A"
    print("Excellent work!")
elif score >= 80:
    grade = "B"
    print("Good job!")
elif score >= 70:
    grade = "C"
    print("Satisfactory.")
elif score >= 60:
    grade = "D"
    print("Needs improvement.")
else:
    grade = "F"
    print("Failed.")

print(f"Your grade is: {grade}")

Note: Only the first true condition executes.

Comparison Operators

Common comparison operators used in conditions:

x = 10
y = 20

print(x == y)   # Equal to: False
print(x != y)   # Not equal to: True
print(x < y)    # Less than: True
print(x > y)    # Greater than: False
print(x <= y)   # Less than or equal to: True
print(x >= y)   # Greater than or equal to: False

Logical Operators

Combine multiple conditions:

age = 25
has_license = True

# AND: Both conditions must be true
if age >= 18 and has_license:
    print("You can drive legally.")

# OR: At least one condition must be true
if age < 18 or not has_license:
    print("You cannot drive.")

# NOT: Negates the condition
if not has_license:
    print("You need a license to drive.")

Nested if Statements

If statements can be nested inside other if statements:

age = 20
has_id = True

if age >= 18:
    if has_id:
        print("You can enter the club.")
        print("Have a great time!")
    else:
        print("You need ID to enter.")
else:
    print("You must be 18 or older.")

Truthy and Falsy Values

Python considers these values falsy:

  • False
  • None
  • 0 (zero)
  • Empty collections: [], (), {}, ""

All other values are truthy:

# Examples of truthy/falsy
if 0:
    print("This won't print")
else:
    print("0 is falsy")

if "hello":
    print("Non-empty string is truthy")

if []:
    print("This won't print")
else:
    print("Empty list is falsy")

Ternary Operator

A compact way to write simple if-else statements:

# Traditional if-else
age = 20
status = ""
if age >= 18:
    status = "Adult"
else:
    status = "Minor"

# Ternary operator
status = "Adult" if age >= 18 else "Minor"

print(f"Status: {status}")

# Can be used in expressions
message = f"You are {'an adult' if age >= 18 else 'a minor'}."

Mini Project

Create a comprehensive grading system program:

  • Accept a numerical score from the user
  • Use if-elif-else to determine the letter grade
  • Include plus/minus grades for more precision
  • Provide encouraging messages based on performance
  • Handle invalid input (scores outside 0-100)
  • Bonus: Calculate GPA or provide study recommendations

Here's a starting template:

def get_grade(score):
    """Convert numerical score to letter grade"""
    if score >= 97:
        return "A+"
    elif score >= 93:
        return "A"
    elif score >= 90:
        return "A-"
    elif score >= 87:
        return "B+"
    elif score >= 83:
        return "B"
    elif score >= 80:
        return "B-"
    elif score >= 77:
        return "C+"
    elif score >= 73:
        return "C"
    elif score >= 70:
        return "C-"
    elif score >= 67:
        return "D+"
    elif score >= 63:
        return "D"
    elif score >= 60:
        return "D-"
    else:
        return "F"

def get_message(grade):
    """Provide feedback based on grade"""
    if grade.startswith("A"):
        return "Excellent work! Keep it up!"
    elif grade.startswith("B"):
        return "Good job! Room for improvement."
    elif grade.startswith("C"):
        return "Satisfactory. Consider extra study."
    elif grade.startswith("D"):
        return "Needs improvement. Seek help if needed."
    else:
        return "Failed. Don't give up - try again!"

# Main program
try:
    score = float(input("Enter your score (0-100): "))
    
    if 0 <= score <= 100:
        grade = get_grade(score)
        message = get_message(grade)
        
        print(f"Score: {score}")
        print(f"Grade: {grade}")
        print(f"Feedback: {message}")
    else:
        print("Invalid score. Please enter a number between 0 and 100.")
        
except ValueError:
    print("Invalid input. Please enter a numerical score.")

Common Mistakes

  • Forgetting colons after conditions
  • Incorrect indentation (Python is indentation-sensitive)
  • Using assignment (=) instead of comparison (==)
  • Missing else clause when needed
  • Overusing nested if statements (consider elif)
  • Not handling edge cases

Best Practices

  • Keep conditions simple and readable
  • Use elif for mutually exclusive conditions
  • Avoid deep nesting when possible
  • Use descriptive variable names
  • Comment complex conditions
  • Test boundary conditions
