Data Types

Data types in Python define the kind of data that can be stored and manipulated in variables. Python has several built-in data types that are automatically determined when you assign a value to a variable. Understanding data types is fundamental to writing effective Python programs, as different types support different operations and have different characteristics.

What are Data Types?

Data types specify:

  • The type of data a variable can hold
  • The operations that can be performed on the data
  • How the data is stored in memory
  • The range and precision of values
  • How Python interprets the data

Basic Data Types

Data Type Description Example Mutable?
int Integer numbers 42, -10 No
float Floating-point numbers 3.14, -2.5 No
str String (text) "Hello", 'Python' No
bool Boolean (True/False) True, False No
list Ordered collection [1, 2, 3] Yes
tuple Immutable ordered collection (1, 2, 3) No
dict Key-value pairs {"key": "value"} Yes
set Unordered unique collection {1, 2, 3} Yes

Type Conversion

Convert between data types using built-in functions:

# String to integer
age_str = "25"
age_int = int(age_str)
print(age_int, type(age_int))  # 25 

# Integer to string
count = 42
count_str = str(count)
print(count_str, type(count_str))  # "42" 

# String to float
price_str = "19.99"
price_float = float(price_str)
print(price_float, type(price_float))  # 19.99 

# Float to integer (truncates)
pi = 3.14159
pi_int = int(pi)
print(pi_int)  # 3

# Integer to boolean
zero = 0

print(bool(zero))     # False
print(bool(non_zero)) # True

# List to tuple
my_list = [1, 2, 3]
my_tuple = tuple(my_list)
print(my_tuple, type(my_tuple))  # (1, 2, 3) 

# String to list
text = "hello"
char_list = list(text)
print(char_list)  # ['h', 'e', 'l', 'l', 'o']

Mutable vs Immutable Types

Understanding mutability is crucial in Python:

# Immutable types (cannot be changed after creation)
x = 42
y = x
x = 43
print(y)  # Still 42 - integers are immutable

s1 = "hello"
s2 = s1
s1 = s1 + " world"
print(s2)  # Still "hello" - strings are immutable

# Mutable types (can be modified)
list1 = [1, 2, 3]
list2 = list1
list1.append(4)
print(list2)  # [1, 2, 3, 4] - both variables reference the same list

# To create a copy of mutable objects
list3 = [1, 2, 3]
list4 = list3.copy()  # or list3[:]
list3.append(4)
print(list4)  # [1, 2, 3] - list4 is a separate copy

Sets

# Creating sets
empty_set = set()  # Note: {} creates a dict
numbers = {1, 2, 3, 4, 5}
mixed_set = {1, "hello", 3.14}

# Set operations
numbers.add(6)      # Add element
numbers.remove(3)   # Remove element
numbers.discard(10) # Remove if exists (no error if not)

# Set operations
set1 = {1, 2, 3, 4}
set2 = {3, 4, 5, 6}

print(set1 | set2)  # Union: {1, 2, 3, 4, 5, 6}
print(set1 & set2)  # Intersection: {3, 4}
print(set1 - set2)  # Difference: {1, 2}
print(set1 ^ set2)  # Symmetric difference: {1, 2, 5, 6}

# Set methods
print(3 in set1)    # True
print(len(set1))    # 4

Mini Project

Create a data type analyzer and converter program:

  • Accept user input of different types
  • Analyze and display the data type of each input
  • Convert between different data types safely
  • Handle conversion errors gracefully
  • Create collections (lists, tuples, sets, dicts) from user input
  • Display information about each data type
  • Bonus: Implement a simple calculator that works with different numeric types

Here's a starting template:

def analyze_data_type(value):
    """Analyze and return information about a data type"""
    data_type = type(value).__name__
    
    info = {
        "type": data_type,
        "value": repr(value),
        "length": None,
        "is_mutable": False,
        "category": "primitive"
    }
    
    # Check length for sequences and mappings
    if hasattr(value, '__len__'):
        info["length"] = len(value)
    
    # Check mutability
    try:
        if isinstance(value, (list, dict, set)):
            info["is_mutable"] = True
            info["category"] = "collection"
        elif isinstance(value, tuple):
            info["category"] = "collection"
        elif isinstance(value, str):
            info["category"] = "text"
        elif isinstance(value, (int, float, complex)):
            info["category"] = "numeric"
        elif isinstance(value, bool):
            info["category"] = "boolean"
    except:
        pass
    
    return info

def safe_convert(value, target_type):
    """Safely convert value to target type"""
    try:
        if target_type == "int":
            return int(float(value)) if isinstance(value, str) else int(value)
        elif target_type == "float":
            return float(value)
        elif target_type == "str":
            return str(value)
        elif target_type == "bool":
            return bool(value)
        elif target_type == "list":
            return list(value) if hasattr(value, '__iter__') else [value]
        elif target_type == "tuple":
            return tuple(value) if hasattr(value, '__iter__') else (value,)
        elif target_type == "set":
            return set(value) if hasattr(value, '__iter__') else {value}
        else:
            return value
    except (ValueError, TypeError) as e:
        return f"Conversion error: {e}"

def create_collection_from_input(inputs, collection_type):
    """Create a collection from multiple inputs"""
    try:
        if collection_type == "list":
            return list(inputs)
        elif collection_type == "tuple":
            return tuple(inputs)
        elif collection_type == "set":
            return set(inputs)
        elif collection_type == "dict":
            # Try to create key-value pairs
            result = {}
            for item in inputs:
                if isinstance(item, str) and ":" in item:
                    key, value = item.split(":", 1)
                    result[key.strip()] = value.strip()
                else:
                    result[str(item)] = item
            return result
        else:
            return inputs
    except Exception as e:
        return f"Error creating collection: {e}"

def display_type_info(info):
    """Display formatted type information"""
    print(f"\nData Type Analysis:")
    print(f"  Type: {info['type']}")
    print(f"  Category: {info['category']}")
    print(f"  Value: {info['value']}")
    if info['length'] is not None:
        print(f"  Length: {info['length']}")
    print(f"  Mutable: {'Yes' if info['is_mutable'] else 'No'}")

def main():
    print("Welcome to the Data Type Analyzer!")
    
    # Store user inputs for collection creation
    user_inputs = []
    
    while True:
        print("\n" + "="*40)
        print("    DATA TYPE ANALYZER")
        print("="*40)
        print("1. Analyze data type")
        print("2. Convert data type")
        print("3. Create collection")
        print("4. View all inputs")
        print("5. Exit")
        print("="*40)
        
        choice = input("Enter your choice (1-5): ").strip()
        
        try:
            if choice == "1":
                user_input = input("Enter a value to analyze: ")
                
                # Try to evaluate as Python literal
                try:
                    value = eval(user_input)
                except:
                    value = user_input
                
                info = analyze_data_type(value)
                display_type_info(info)
                user_inputs.append(value)
                
            elif choice == "2":
                if not user_inputs:
                    print("No inputs available. Please analyze some data first.")
                    continue
                    
                print("Available inputs:")
                for i, inp in enumerate(user_inputs, 1):
                    print(f"  {i}. {repr(inp)} ({type(inp).__name__})")
                
                try:
                    index = int(input("Select input number: ")) - 1
                    value = user_inputs[index]
                except (ValueError, IndexError):
                    print("Invalid selection.")
                    continue
                
                print(f"Selected: {repr(value)} ({type(value).__name__})")
                
                target_types = ["int", "float", "str", "bool", "list", "tuple", "set"]
                print("Available target types:")
                for i, t in enumerate(target_types, 1):
                    print(f"  {i}. {t}")
                
                try:
                    type_index = int(input("Select target type number: ")) - 1
                    target_type = target_types[type_index]
                except (ValueError, IndexError):
                    print("Invalid selection.")
                    continue
                
                converted = safe_convert(value, target_type)
                print(f"Converted to {target_type}: {repr(converted)} ({type(converted).__name__})")
                
            elif choice == "3":
                if len(user_inputs) < 2:
                    print("Need at least 2 inputs to create a collection.")
                    continue
                
                collection_types = ["list", "tuple", "set", "dict"]
                print("Available collection types:")
                for i, t in enumerate(collection_types, 1):
                    print(f"  {i}. {t}")
                
                try:
                    type_index = int(input("Select collection type number: ")) - 1
                    collection_type = collection_types[type_index]
                except (ValueError, IndexError):
                    print("Invalid selection.")
                    continue
                
                collection = create_collection_from_input(user_inputs, collection_type)
                print(f"Created {collection_type}: {repr(collection)}")
                
                # Analyze the collection
                info = analyze_data_type(collection)
                display_type_info(info)
                
            elif choice == "4":
                if not user_inputs:
                    print("No inputs recorded yet.")
                else:
                    print("All recorded inputs:")
                    for i, inp in enumerate(user_inputs, 1):
                        info = analyze_data_type(inp)
                        print(f"  {i}. {info['value']} ({info['type']})")
                        
            elif choice == "5":
                print("Thank you for using the Data Type Analyzer!")
                break
                
            else:
                print("Invalid choice. Please select 1-5.")
                
        except Exception as e:
            print(f"An error occurred: {e}")
        
        input("\nPress Enter to continue...")

if __name__ == "__main__":
    main()

Common Mistakes

  • Confusing mutable and immutable types
  • Not understanding automatic type conversion
  • Using wrong type conversion functions
  • Attempting to modify immutable objects
  • Mixing data types inappropriately
  • Not checking types before operations
  • Forgetting that sets and dicts are unordered

Best Practices

  • Use descriptive variable names that indicate type
  • Check types when necessary with isinstance()
  • Use appropriate data structures for your needs
  • Consider memory usage for large collections
  • Document expected types in functions
  • Use type hints (Python 3.5+) for better code clarity
  • Handle type conversion errors gracefully
