While Loops

While loops in Python are used to execute a block of code repeatedly as long as a specified condition is true. Unlike for loops, which iterate over a sequence, while loops continue executing until the condition becomes false. This makes them ideal for situations where you don't know in advance how many times the loop should run.

The basic syntax of a while loop is:

while condition:
    # code to execute

The condition is evaluated before each iteration. If it's true, the code block runs; if false, the loop ends.

While loops can be used for various tasks such as reading user input until a valid response is given, processing data until a certain condition is met, or implementing game loops.

Examples

# Basic while loop
count = 0
while count < 5:
    print(f"Count: {count}")
    count += 1

print("Loop finished!")

# While loop with user input
user_input = ""
while user_input != "quit":
    user_input = input("Enter something (type 'quit' to exit): ")
    print(f"You entered: {user_input}")

# While loop with break
number = 0
while True:
    print(f"Number: {number}")
    number += 1
    if number >= 5:
        break

This example shows different ways to use while loops: a basic counter, reading input until a condition is met, and using break to exit the loop.
