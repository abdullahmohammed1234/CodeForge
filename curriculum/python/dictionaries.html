Dictionaries

Dictionaries in Python are powerful data structures that store key-value pairs. Unlike lists that are indexed by numbers, dictionaries allow you to access values using meaningful keys. They are mutable, unordered (in Python 3.7+ they maintain insertion order), and extremely versatile for organizing and manipulating data.

What are Dictionaries?

Dictionaries provide fast lookups and are ideal for:

  • Storing related data with descriptive keys
  • Creating mappings between different types of data
  • Representing real-world objects with properties
  • Caching results for quick access
  • Counting occurrences of items
  • Configuration settings and parameters

Creating Dictionaries

Several ways to create dictionaries:

# Empty dictionary
empty_dict = {}

# Dictionary with initial values
person = {
    "name": "Alice",
    "age": 25,
    "city": "New York"
}

# Using dict() constructor
student = dict(name="Bob", age=20, grade="A")

# Mixed data types
mixed_dict = {
    "string": "hello",
    "number": 42,
    "list": [1, 2, 3],
    "boolean": True
}

print(person)
print(student)
print(mixed_dict)

Accessing Dictionary Values

Use keys to access values:

person = {"name": "Alice", "age": 25, "city": "New York"}

# Access using square brackets
print(person["name"])    # Alice
print(person["age"])     # 25

# Using get() method (safer)
print(person.get("city"))     # New York
print(person.get("salary"))   # None (key doesn't exist)
print(person.get("salary", "Not specified"))  # Default value

Modifying Dictionaries

Dictionaries are mutable - you can change them:

person = {"name": "Alice", "age": 25}

# Add new key-value pair
person["city"] = "New York"
print(person)  # {'name': 'Alice', 'age': 25, 'city': 'New York'}

# Modify existing value
person["age"] = 26
print(person)  # {'name': 'Alice', 'age': 26, 'city': 'New York'}

# Update multiple values
person.update({"age": 27, "email": "alice@example.com"})
print(person)  # {'name': 'Alice', 'age': 27, 'city': 'New York', 'email': 'alice@example.com'}

Dictionary Methods

Essential dictionary methods:

person = {"name": "Alice", "age": 25, "city": "New York"}

# Get all keys
print(person.keys())    # dict_keys(['name', 'age', 'city'])

# Get all values
print(person.values())  # dict_values(['Alice', 25, 'New York'])

# Get all key-value pairs
print(person.items())   # dict_items([('name', 'Alice'), ('age', 25), ('city', 'New York')])

# Remove and return a value
age = person.pop("age")
print(age)        # 25
print(person)     # {'name': 'Alice', 'city': 'New York'}

# Remove last inserted item
last_item = person.popitem()
print(last_item)  # ('city', 'New York')

# Clear all items
person.clear()
print(person)     # {}

Checking Key Existence

Check if keys exist before accessing:

person = {"name": "Alice", "age": 25}

# Using 'in' operator
if "name" in person:
    print("Name:", person["name"])

if "salary" not in person:
    print("Salary not specified")

# Check for multiple keys
required_keys = ["name", "age"]
if all(key in person for key in required_keys):
    print("All required information present")

Iterating Over Dictionaries

Different ways to iterate:

student = {"name": "Bob", "age": 20, "grade": "A", "major": "Computer Science"}

# Iterate over keys
print("Keys:")
for key in student:
    print(key)

# Iterate over values
print("\nValues:")
for value in student.values():
    print(value)

# Iterate over key-value pairs
print("\nKey-Value pairs:")
for key, value in student.items():
    print(f"{key}: {value}")

# Iterate with enumeration
print("\nWith index:")
for i, (key, value) in enumerate(student.items(), 1):
    print(f"{i}. {key}: {value}")

Dictionary Comprehensions

Create dictionaries using comprehensions:

# Create a dictionary of squares
squares = {x: x**2 for x in range(1, 6)}
print(squares)  # {1: 1, 2: 4, 3: 9, 4: 16, 5: 25}

# Filter dictionary
grades = {"Alice": 85, "Bob": 92, "Charlie": 78, "Diana": 95}
passing_grades = {name: grade for name, grade in grades.items() if grade >= 80}
print(passing_grades)  # {'Alice': 85, 'Bob': 92, 'Diana': 95}

# Transform values
uppercase_names = {name.upper(): grade for name, grade in grades.items()}
print(uppercase_names)  # {'ALICE': 85, 'BOB': 92, 'CHARLIE': 78, 'DIANA': 95}

Nested Dictionaries

Dictionaries can contain other dictionaries:

company = {
    "name": "Tech Corp",
    "departments": {
        "engineering": {
            "manager": "Alice",
            "employees": ["Bob", "Charlie"],
            "budget": 500000
        },
        "sales": {
            "manager": "Diana",
            "employees": ["Eve", "Frank"],
            "budget": 300000
        }
    }
}

# Access nested values
print(company["departments"]["engineering"]["manager"])  # Alice

# Add to nested dictionary
company["departments"]["engineering"]["employees"].append("Grace")
print(company["departments"]["engineering"]["employees"])  # ['Bob', 'Charlie', 'Grace']

Dictionary Operations

Advanced operations:

dict1 = {"a": 1, "b": 2}
dict2 = {"c": 3, "d": 4}

# Merge dictionaries (Python 3.9+)
merged = dict1 | dict2
print(merged)  # {'a': 1, 'b': 2, 'c': 3, 'd': 4}

# Or using update
dict1_copy = dict1.copy()
dict1_copy.update(dict2)
print(dict1_copy)  # {'a': 1, 'b': 2, 'c': 3, 'd': 4}

# Set default values
person = {}
person.setdefault("name", "Unknown")
person.setdefault("age", 0)
print(person)  # {'name': 'Unknown', 'age': 0}

# Count occurrences
words = ["apple", "banana", "apple", "cherry", "banana", "apple"]
word_count = {}
for word in words:
    word_count[word] = word_count.get(word, 0) + 1
print(word_count)  # {'apple': 3, 'banana': 2, 'cherry': 1}

Dictionary vs Other Data Structures

Data Structure Use Case Access Method
List Ordered collection of items Index (position)
Dictionary Key-value mappings Key (unique identifier)
Set Unique items, membership testing No indexing
Tuple Immutable ordered collection Index (position)

Mini Project

Create a student management system using dictionaries:

  • Create a dictionary to store student information
  • Implement functions to add, remove, and update students
  • Calculate class statistics (average grades, top performers)
  • Search and filter students based on criteria
  • Save and load student data (bonus)
  • Use nested dictionaries for complex data structures

Here's a starting template:

def add_student(students, student_id, name, age, grade, major):
    """Add a new student to the system"""
    students[student_id] = {
        "name": name,
        "age": age,
        "grade": grade,
        "major": major
    }

def remove_student(students, student_id):
    """Remove a student from the system"""
    if student_id in students:
        removed_student = students.pop(student_id)
        print(f"Removed student: {removed_student['name']}")
        return True
    else:
        print("Student not found")
        return False

def update_student_grade(students, student_id, new_grade):
    """Update a student's grade"""
    if student_id in students:
        students[student_id]["grade"] = new_grade
        print(f"Updated {students[student_id]['name']}'s grade to {new_grade}")
        return True
    else:
        print("Student not found")
        return False

def calculate_class_average(students):
    """Calculate the average grade of all students"""
    if not students:
        return 0
    
    total_grade = sum(student["grade"] for student in students.values())
    return total_grade / len(students)

def find_top_performers(students, top_n=3):
    """Find the top performing students"""
    # Sort students by grade in descending order
    sorted_students = sorted(students.items(), 
                           key=lambda x: x[1]["grade"], 
                           reverse=True)
    return sorted_students[:top_n]

def search_students_by_major(students, major):
    """Find all students in a specific major"""
    return {sid: info for sid, info in students.items() 
            if info["major"].lower() == major.lower()}

def display_student_info(students, student_id):
    """Display information for a specific student"""
    if student_id in students:
        student = students[student_id]
        print(f"ID: {student_id}")
        print(f"Name: {student['name']}")
        print(f"Age: {student['age']}")
        print(f"Grade: {student['grade']}")
        print(f"Major: {student['major']}")
    else:
        print("Student not found")

def display_all_students(students):
    """Display information for all students"""
    if not students:
        print("No students in the system")
        return
    
    print(f"\nTotal students: {len(students)}")
    print("-" * 50)
    
    for student_id, info in students.items():
        print(f"ID: {student_id} | Name: {info['name']} | Grade: {info['grade']} | Major: {info['major']}")

# Main program
def main():
    students = {}
    
    # Add some sample students
    add_student(students, "S001", "Alice Johnson", 20, 85, "Computer Science")
    add_student(students, "S002", "Bob Smith", 21, 92, "Engineering")
    add_student(students, "S003", "Charlie Brown", 19, 78, "Mathematics")
    add_student(students, "S004", "Diana Prince", 22, 95, "Computer Science")
    add_student(students, "S005", "Eve Wilson", 20, 88, "Engineering")
    
    while True:
        print("\n" + "="*40)
        print("    STUDENT MANAGEMENT SYSTEM")
        print("="*40)
        print("1. Display all students")
        print("2. Add new student")
        print("3. Remove student")
        print("4. Update student grade")
        print("5. View student details")
        print("6. Calculate class average")
        print("7. Show top performers")
        print("8. Search by major")
        print("9. Exit")
        print("="*40)
        
        choice = input("Enter your choice (1-9): ").strip()
        
        if choice == "1":
            display_all_students(students)
            
        elif choice == "2":
            student_id = input("Enter student ID: ").strip()
            name = input("Enter student name: ").strip()
            age = int(input("Enter age: "))
            grade = float(input("Enter grade: "))
            major = input("Enter major: ").strip()
            add_student(students, student_id, name, age, grade, major)
            
        elif choice == "3":
            student_id = input("Enter student ID to remove: ").strip()
            remove_student(students, student_id)
            
        elif choice == "4":
            student_id = input("Enter student ID: ").strip()
            new_grade = float(input("Enter new grade: "))
            update_student_grade(students, student_id, new_grade)
            
        elif choice == "5":
            student_id = input("Enter student ID: ").strip()
            display_student_info(students, student_id)
            
        elif choice == "6":
            avg = calculate_class_average(students)
            print(f"Class average: {avg:.2f}")
            
        elif choice == "7":
            top_students = find_top_performers(students)
            print("Top performers:")
            for i, (sid, info) in enumerate(top_students, 1):
                print(f"{i}. {info['name']} (ID: {sid}) - Grade: {info['grade']}")
            
        elif choice == "8":
            major = input("Enter major to search: ").strip()
            matching_students = search_students_by_major(students, major)
            if matching_students:
                print(f"Students in {major}:")
                for sid, info in matching_students.items():
                    print(f"  {info['name']} (ID: {sid})")
            else:
                print(f"No students found in {major}")
            
        elif choice == "9":
            print("Thank you for using the Student Management System!")
            break
            
        else:
            print("Invalid choice. Please select 1-9.")
        
        input("\nPress Enter to continue...")

if __name__ == "__main__":
    main()

Common Mistakes

  • Using lists when dictionaries would be more appropriate
  • Forgetting that keys must be immutable (strings, numbers, tuples)
  • Attempting to access non-existent keys without checking
  • Modifying dictionary size while iterating over it
  • Using mutable objects as keys
  • Not understanding that dictionaries maintain insertion order (Python 3.7+)

Performance Considerations

  • Dictionary lookups are O(1) on average
  • Use dictionaries for frequent lookups
  • Consider memory usage for large dictionaries
  • Use dict.get() for safe key access
  • Avoid frequent dictionary resizing
