Getting Started with Python

Getting started with Python is straightforward and beginner-friendly. This guide will walk you through installing Python, setting up your development environment, and writing your first program. Whether you're completely new to programming or switching from another language, Python's simplicity makes it an excellent choice for your first steps.

System Requirements

Python runs on virtually any modern computer:

  • Operating Systems: Windows, macOS, Linux, and even some mobile devices
  • Hardware: Minimal requirements - works on most computers from the last decade
  • Storage: About 100MB for installation plus space for your projects
  • Internet: Required for installation and accessing online resources

Installing Python

The easiest way to install Python is through the official Python website:

  1. Visit python.org
  2. Go to the Downloads section
  3. Download the latest stable version for your operating system
  4. Run the installer and follow the setup wizard
  5. Make sure to check "Add Python to PATH" during installation

Pro Tip: If you're on Windows, consider using the Microsoft Store version of Python for easier updates.

Verifying Installation

After installation, verify Python is working:

# Open your command prompt/terminal and type:
python --version
# or
python3 --version

# You should see something like:
Python 3.11.5

Choosing a Code Editor

While you can write Python code in any text editor, a good code editor enhances your experience:

  • Visual Studio Code (VS Code): Free, powerful, excellent Python support
  • PyCharm: Professional IDE with community edition available
  • IDLE: Comes with Python, good for beginners
  • Sublime Text: Lightweight and fast
  • Atom: Customizable and free

Setting Up VS Code for Python

If you choose VS Code:

  1. Download and install VS Code from code.visualstudio.com
  2. Open VS Code and go to Extensions (Ctrl+Shift+X)
  3. Search for "Python" and install the official Python extension by Microsoft
  4. Install the "Pylint" extension for code linting

Your First Python Program

Let's create your first Python program:

  1. Create a new file called hello.py
  2. Type the following code:
print("Hello, World!")
print("Welcome to Python programming!")
  1. Save the file
  2. Open your terminal/command prompt
  3. Navigate to the folder containing your file
  4. Run the program: python hello.py

Understanding the Code

  • print() is a built-in function that displays text
  • Text in quotes is called a string
  • Each print() statement outputs on a new line
  • The program runs from top to bottom

Interactive Python (REPL)

Python comes with an interactive shell for quick testing:

# Type 'python' in your terminal to start
>>> print("Hello from REPL!")
Hello from REPL!
>>> 2 + 3
5
>>> name = "Python"
>>> print(f"Hello, {name}!")
Hello, Python!

Use Ctrl+Z (Windows) or Ctrl+D (macOS/Linux) to exit.

Mini Project

Create a personalized welcome program:

  • Create a Python file that introduces yourself
  • Use multiple print statements
  • Include your name, favorite programming language, and goals
  • Experiment with different ways to format your output
  • Bonus: Use variables to store your information

Here's a starting template:

# My first Python program
name = "Your Name"
language = "Python"
goal = "to build amazing applications"

print("Hello, World!")
print(f"My name is {name}")
print(f"I'm learning {language}")
print(f"My goal is {goal}")
print("Let's start coding!")

Common Installation Issues

  • "python is not recognized": Make sure Python is added to PATH
  • Permission errors: Try running terminal as administrator
  • Multiple Python versions: Use python3 instead of python
  • VS Code not recognizing Python: Reload VS Code after installing extensions
