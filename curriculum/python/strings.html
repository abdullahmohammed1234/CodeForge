Strings

Strings are sequences of characters in Python. They are immutable, meaning they cannot be changed after creation. Strings can be created using single quotes, double quotes, or triple quotes for multi-line strings.

Key characteristics of strings:

  • Immutable: Cannot be modified in place
  • Ordered: Characters have positions
  • Indexable: Access characters by index
  • Sliceable: Extract substrings

Strings support various operations like concatenation, repetition, and many built-in methods for manipulation.

Examples

# Creating strings
single_quote = 'Hello'
double_quote = "World"
multi_line = """This is a
multi-line string"""

# Accessing characters
print(single_quote[0])  # 'H'
print(double_quote[-1])  # 'd'

# Slicing
print(single_quote[1:4])  # 'ell'
print(double_quote[::-1])  # 'dlroW' (reversed)

# String operations
greeting = single_quote + " " + double_quote  # 'Hello World'
repeated = single_quote * 3  # 'HelloHelloHello'

# String methods
upper_case = single_quote.upper()  # 'HELLO'
lower_case = double_quote.lower()  # 'world'
capitalized = single_quote.capitalize()  # 'Hello'
title_case = "hello world".title()  # 'Hello World'

# Finding and replacing
position = greeting.find("World")  # 6
replaced = greeting.replace("World", "Python")  # 'Hello Python'

# Splitting and joining
words = greeting.split()  # ['Hello', 'World']
joined = "-".join(words)  # 'Hello-World'

# Checking properties
print(greeting.startswith("Hello"))  # True
print(greeting.endswith("World"))  # True
print(greeting.isalpha())  # False (contains space)
print(single_quote.isalpha())  # True

This example demonstrates creating strings, accessing and manipulating them, and using various string methods.

Mini Project

Create a text analyzer program that takes a sentence as input and provides various statistics and manipulations.

  • Ask the user to enter a sentence
  • Count the number of words, characters, and vowels
  • Convert the sentence to uppercase and lowercase
  • Reverse the sentence
  • Replace all vowels with asterisks
  • Bonus: Check if the sentence is a palindrome

Here's a starting template:

sentence = input("Enter a sentence: ")

# Count words
word_count = len(sentence.split())

# Count characters
char_count = len(sentence)

# Count vowels
vowels = "aeiouAEIOU"
vowel_count = sum(1 for char in sentence if char in vowels)

print(f"Words: {word_count}")
print(f"Characters: {char_count}")
print(f"Vowels: {vowel_count}")
print(f"Uppercase: {sentence.upper()}")
print(f"Lowercase: {sentence.lower()}")
print(f"Reversed: {sentence[::-1]}")

# Replace vowels
no_vowels = "".join("*" if char in vowels else char for char in sentence)
print(f"No vowels: {no_vowels}")
