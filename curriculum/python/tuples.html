Tuples

Tuples in Python are immutable sequences of objects. They are similar to lists but cannot be modified after creation. Tuples are useful for storing related data that shouldn't be changed, and they can be used as dictionary keys.

What are Tuples?

  • Immutable sequences of objects
  • Ordered collections (maintain insertion order)
  • Can contain elements of different types
  • Defined using parentheses () or comma separation
  • Faster than lists for iteration
  • Can be used as dictionary keys (unlike lists)
  • Support indexing and slicing

Creating Tuples

# Empty tuple
empty_tuple = ()

# Single element tuple (note the comma)
single_tuple = (42,)
single_tuple2 = 42,  # Also valid

# Multiple elements
numbers = (1, 2, 3, 4, 5)
fruits = ("apple", "banana", "cherry")
mixed = (1, "hello", 3.14, True)

# Nested tuples
nested = ((1, 2), (3, 4), (5, 6))

# Tuple from other iterables
list_to_tuple = tuple([1, 2, 3, 4])
string_to_tuple = tuple("hello")  # ('h', 'e', 'l', 'l', 'o')

print(numbers)      # (1, 2, 3, 4, 5)
print(fruits)       # ('apple', 'banana', 'cherry')
print(mixed)        # (1, 'hello', 3.14, True)
print(string_to_tuple)  # ('h', 'e', 'l', 'l', 'o')

Accessing Tuple Elements

fruits = ("apple", "banana", "cherry", "orange", "grape")

# Indexing (zero-based)
print(fruits[0])    # 'apple'
print(fruits[2])    # 'cherry'
print(fruits[-1])   # 'grape' (last element)
print(fruits[-2])   # 'orange' (second to last)

# Slicing
print(fruits[1:4])  # ('banana', 'cherry', 'orange')
print(fruits[:3])   # ('apple', 'banana', 'cherry')
print(fruits[2:])   # ('cherry', 'orange', 'grape')
print(fruits[::2])  # ('apple', 'cherry', 'grape') - every other element

# Length
print(len(fruits))  # 5

Tuple Immutability

numbers = (1, 2, 3, 4, 5)

# This will cause an error
# numbers[0] = 10  # TypeError: 'tuple' object does not support item assignment

# But you can create new tuples
new_numbers = numbers[:2] + (10,) + numbers[3:]
print(new_numbers)  # (1, 2, 10, 4, 5)

# If tuple contains mutable objects, they can be modified
mutable_tuple = ([1, 2], [3, 4])
mutable_tuple[0].append(99)
print(mutable_tuple)  # ([1, 2, 99], [3, 4])

Tuple Operations

tuple1 = (1, 2, 3)
tuple2 = (4, 5, 6)

# Concatenation
combined = tuple1 + tuple2
print(combined)  # (1, 2, 3, 4, 5, 6)

# Repetition
repeated = tuple1 * 3
print(repeated)  # (1, 2, 3, 1, 2, 3, 1, 2, 3)

# Membership testing
print(2 in tuple1)      # True
print(6 in tuple1)      # False

# Counting occurrences
numbers = (1, 2, 2, 3, 2, 4)
print(numbers.count(2))  # 3

# Finding index
print(numbers.index(3))  # 3
# print(numbers.index(5))  # ValueError: 5 is not in tuple

Tuple Unpacking

# Basic unpacking
point = (3, 4)
x, y = point
print(f"x = {x}, y = {y}")  # x = 3, y = 4

# Unpacking with extended iterable unpacking (Python 3)
numbers = (1, 2, 3, 4, 5)
first, *middle, last = numbers
print(first)   # 1
print(middle)  # [2, 3, 4]
print(last)    # 5

# Swapping values
a, b = 5, 10
a, b = b, a
print(f"a = {a}, b = {b}")  # a = 10, b = 5

# Ignoring values
name, _, age, _ = ("Alice", "female", 25, "engineer")
print(f"{name} is {age} years old")  # Alice is 25 years old

Tuple Methods

numbers = (1, 2, 3, 2, 4, 2, 5)

# count() - count occurrences of a value
print(numbers.count(2))  # 3

# index() - find first occurrence of a value
print(numbers.index(4))  # 4
print(numbers.index(2))  # 1 (first occurrence)

# len() - get length
print(len(numbers))  # 7

# max() and min()
print(max(numbers))  # 5
print(min(numbers))  # 1

# sum()
print(sum(numbers))  # 19

Tuples vs Lists

Aspect Tuples Lists
Mutability Immutable Mutable
Syntax () []
Performance Faster Slower
Memory usage Less More
Methods Fewer (2) Many
Use as dict keys Yes No

When to Use Tuples

  • Storing related data that shouldn't change
  • Using as dictionary keys
  • Returning multiple values from functions
  • String formatting with % operator
  • Performance-critical code
  • Ensuring data integrity

Mini Project

Create a student record system using tuples:

  • Store student information as tuples (name, age, grades)
  • Create a list of student tuples
  • Implement functions to find top students, calculate averages
  • Sort students by different criteria
  • Display formatted student reports
  • Bonus: Add search functionality by name or grade range

Here's a starting template:

def calculate_average(grades):
    """Calculate average of grades."""
    return sum(grades) / len(grades) if grades else 0

def find_top_student(students):
    """Find student with highest average grade."""
    top_student = max(students, key=lambda s: calculate_average(s[2]))
    return top_student

def display_student(student):
    """Display formatted student information."""
    name, age, grades = student
    avg = calculate_average(grades)
    print(f"Name: {name}, Age: {age}, Grades: {grades}, Average: {avg:.2f}")

# Sample data
students = [
    ("Alice", 20, (85, 90, 88)),
    ("Bob", 19, (78, 82, 85)),
    ("Charlie", 21, (92, 89, 94)),
    ("Diana", 20, (88, 91, 87))
]

print("All students:")
for student in students:
    display_student(student)

print("\nTop student:")
top = find_top_student(students)
display_student(top)

# Sort by average grade
sorted_students = sorted(students, key=lambda s: calculate_average(s[2]), reverse=True)
print("\nStudents sorted by average grade:")
for student in sorted_students:
    display_student(student)
