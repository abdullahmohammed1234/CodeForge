Comments

Comments in Python are pieces of text that are ignored by the Python interpreter. They are used to explain code, make it more readable, and provide documentation. Good commenting practices are essential for writing maintainable and collaborative code.

Why Use Comments?

  • Explain what code does and why
  • Make code easier to understand for others (and future you)
  • Document function purposes, parameters, and return values
  • Temporarily disable code during debugging
  • Provide context for complex algorithms
  • Mark TODO items and known issues

Single-Line Comments

Single-line comments start with the # symbol and continue to the end of the line:

# This is a single-line comment
print("Hello, World!")  # This comment is inline

# Comments can be on their own line
# or at the end of a line of code

# Use comments to explain complex logic
x = 42  # The answer to life, the universe, and everything

# Comments are completely ignored by Python
# print("This line won't execute")

Multi-Line Comments

Python doesn't have a built-in multi-line comment syntax, but you can use multiple single-line comments or triple-quoted strings:

# Method 1: Multiple single-line comments
# This is a multi-line comment
# that spans several lines
# using multiple # symbols

# Method 2: Triple-quoted strings (not true comments, but often used as such)
"""
This is a triple-quoted string
that can span multiple lines
and is often used as a multi-line comment
"""
print("This will execute")

# Method 3: Backslash continuation (not recommended for comments)
# This is a comment that \
# continues on the next line

Docstrings

Docstrings are special comments used to document functions, classes, and modules:

def calculate_area(length, width):
    """
    Calculate the area of a rectangle.
    
    Args:
        length (float): The length of the rectangle
        width (float): The width of the rectangle
        
    Returns:
        float: The area of the rectangle
        
    Example:
        >>> calculate_area(5, 3)
        15
    """
    return length * width

class Rectangle:
    """
    A class representing a rectangle.
    
    Attributes:
        length (float): The length of the rectangle
        width (float): The width of the rectangle
    """
    
    def __init__(self, length, width):
        """
        Initialize a Rectangle object.
        
        Args:
            length (float): The length of the rectangle
            width (float): The width of the rectangle
        """
        self.length = length
        self.width = width

# Accessing docstrings
print(calculate_area.__doc__)
print(Rectangle.__doc__)
print(Rectangle.__init__.__doc__)

Commenting Best Practices

  • Write comments in clear, simple English
  • Explain why, not just what (the code already shows what)
  • Keep comments up to date with code changes
  • Use consistent formatting and style
  • Avoid obvious comments that don't add value
  • Use docstrings for functions and classes
  • Comment complex algorithms and business logic
# Good comments
# Calculate compound interest using the formula A = P(1 + r/n)^(nt)
def compound_interest(principal, rate, time, compounds_per_year):
    return principal * (1 + rate/compounds_per_year) ** (compounds_per_year * time)

# Bad comments (obvious or redundant)
x = 5  # Set x to 5
print("Hello")  # Print Hello

Commenting Code Blocks

def process_data(data):
    """
    Process a list of numbers by filtering, sorting, and calculating statistics.
    """
    # Step 1: Filter out negative numbers
    positive_numbers = [x for x in data if x >= 0]
    
    # Step 2: Sort the numbers in ascending order
    sorted_numbers = sorted(positive_numbers)
    
    # Step 3: Calculate basic statistics
    if sorted_numbers:
        minimum = sorted_numbers[0]
        maximum = sorted_numbers[-1]
        # Use integer division for median calculation
        median_index = len(sorted_numbers) // 2
        median = sorted_numbers[median_index]
        
        return {
            'min': minimum,
            'max': maximum,
            'median': median,
            'count': len(sorted_numbers)
        }
    else:
        return None

# Example usage
data = [3, -1, 7, 2, -5, 8, 4]
result = process_data(data)
print(result)  # {'min': 2, 'max': 8, 'median': 4, 'count': 6}

Commenting Conditional Logic

def get_discount_percentage(customer_type, purchase_amount):
    """
    Calculate discount percentage based on customer type and purchase amount.
    """
    discount = 0
    
    # Premium customers get base discount
    if customer_type == "premium":
        discount = 10
    elif customer_type == "regular":
        discount = 5
    
    # Additional discount for large purchases
    if purchase_amount > 1000:
        discount += 5  # Extra 5% for orders over $1000
    elif purchase_amount > 500:
        discount += 2  # Extra 2% for orders over $500
    
    # Cap discount at 20%
    if discount > 20:
        discount = 20
    
    return discount

# Test the function
print(get_discount_percentage("premium", 1200))  # 15
print(get_discount_percentage("regular", 600))   # 7

PEP 8 and Comment Style

Follow Python's style guide (PEP 8) for comments:

  • Use complete sentences with proper capitalization
  • End sentences with periods
  • Keep line length under 79 characters
  • Use spaces after # for inline comments
  • Align inline comments consistently

Mini Project

Create a well-commented program that manages a simple library system:

  • Create functions for adding books, checking out books, and returning books
  • Use comprehensive docstrings for all functions
  • Add inline comments explaining complex logic
  • Include block comments for different sections of code
  • Create a main function with comments explaining the program flow
  • Bonus: Add input validation with helpful error messages

Here's a starting template:

"""
Simple Library Management System

This program demonstrates good commenting practices by implementing
a basic library system with books, checkouts, and returns.
"""

def add_book(library, title, author):
    """
    Add a new book to the library.
    
    Args:
        library (dict): Dictionary containing library data
        title (str): Title of the book
        author (str): Author of the book
        
    Returns:
        bool: True if book was added, False if it already exists
    """
    # Check if book already exists
    for book in library['books']:
        if book['title'].lower() == title.lower():
            return False  # Book already exists
    
    # Create new book entry
    new_book = {
        'title': title,
        'author': author,
        'available': True,
        'borrower': None
    }
    
    library['books'].append(new_book)
    return True

def checkout_book(library, title, borrower):
    """
    Check out a book to a borrower.
    
    Args:
        library (dict): Dictionary containing library data
        title (str): Title of the book to check out
        borrower (str): Name of the person borrowing the book
        
    Returns:
        bool: True if checkout successful, False otherwise
    """
    # Find the book
    for book in library['books']:
        if book['title'].lower() == title.lower():
            if book['available']:
                # Book is available, check it out
                book['available'] = False
                book['borrower'] = borrower
                return True
            else:
                return False  # Book is already checked out
    
    return False  # Book not found

def return_book(library, title):
    """
    Return a checked out book.
    
    Args:
        library (dict): Dictionary containing library data
        title (str): Title of the book to return
        
    Returns:
        bool: True if return successful, False otherwise
    """
    for book in library['books']:
        if book['title'].lower() == title.lower():
            if not book['available']:
                # Book is checked out, return it
                book['available'] = True
                book['borrower'] = None
                return True
            else:
                return False  # Book was not checked out
    
    return False  # Book not found

def display_library(library):
    """
    Display all books in the library with their status.
    """
    print("\nLibrary Collection:")
    print("-" * 50)
    
    for book in library['books']:
        status = "Available" if book['available'] else f"Checked out by {book['borrower']}"
        print(f"'{book['title']}' by {book['author']} - {status}")

def main():
    """
    Main function demonstrating the library system.
    """
    # Initialize library
    library = {'books': []}
    
    # Add some books
    add_book(library, "Python Crash Course", "Eric Matthes")
    add_book(library, "Clean Code", "Robert C. Martin")
    add_book(library, "The Pragmatic Programmer", "Andrew Hunt")
    
    # Demonstrate checkout and return
    display_library(library)
    
    checkout_book(library, "Clean Code", "Alice")
    display_library(library)
    
    return_book(library, "Clean Code")
    display_library(library)

if __name__ == "__main__":
    main()
