Lists

Lists are one of the most versatile data structures in Python. They are mutable sequences that can hold items of different types, including other lists. Lists are created using square brackets [] and are ordered, meaning the order of elements is preserved.

Key characteristics of lists:

  • Mutable: Can be modified after creation
  • Ordered: Elements maintain their position
  • Heterogeneous: Can contain different data types
  • Dynamic: Can grow or shrink in size

Lists support various operations like indexing, slicing, appending, inserting, removing, and sorting. They are widely used for storing collections of data that may need to be modified.

Examples

# Creating lists
empty_list = []
numbers = [1, 2, 3, 4, 5]
mixed_list = [1, "hello", 3.14, True]
nested_list = [[1, 2], [3, 4], [5, 6]]

# Accessing elements
print(numbers[0])  # 1
print(numbers[-1])  # 5 (last element)

# Slicing
print(numbers[1:4])  # [2, 3, 4]
print(numbers[::2])  # [1, 3, 5] (every other element)

# Modifying lists
numbers.append(6)  # Add to end
numbers.insert(0, 0)  # Insert at beginning
numbers[2] = 10  # Change element
numbers.remove(3)  # Remove first occurrence
popped = numbers.pop()  # Remove and return last element

# List operations
list1 = [1, 2, 3]
list2 = [4, 5, 6]
combined = list1 + list2  # [1, 2, 3, 4, 5, 6]
repeated = list1 * 3  # [1, 2, 3, 1, 2, 3, 1, 2, 3]

# List methods
numbers.sort()  # Sort in place
numbers.reverse()  # Reverse in place
length = len(numbers)  # Get length
index = numbers.index(5)  # Find index of element

This example demonstrates creating lists, accessing and modifying elements, and using various list methods and operations.
