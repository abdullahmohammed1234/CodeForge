Operators

Operators are special symbols in Python that perform operations on variables and values. They allow you to perform mathematical calculations, comparisons, logical operations, and more.

Arithmetic Operators

Arithmetic operators perform mathematical operations:

x = 10
y = 3

print("Addition:", x + y)        # 13
print("Subtraction:", x - y)     # 7
print("Multiplication:", x * y)  # 30
print("Division:", x / y)        # 3.333...
print("Floor Division:", x // y) # 3
print("Modulus:", x % y)         # 1
print("Exponentiation:", x ** y) # 1000

Comparison Operators

Comparison operators compare two values and return True or False:

a = 5
b = 10

print("Equal:", a == b)           # False
print("Not Equal:", a != b)       # True
print("Greater Than:", a > b)     # False
print("Less Than:", a < b)        # True
print("Greater or Equal:", a >= b) # False
print("Less or Equal:", a <= b)   # True

Assignment Operators

Assignment operators assign values to variables:

x = 5

x += 3  # x = x + 3 → x = 8
x -= 2  # x = x - 2 → x = 6
x *= 4  # x = x * 4 → x = 24
x /= 3  # x = x / 3 → x = 8.0
x //= 2 # x = x // 2 → x = 4.0
x %= 3  # x = x % 3 → x = 1.0
x **= 2 # x = x ** 2 → x = 1.0

Logical Operators

Logical operators combine conditional statements:

x = True
y = False

print("AND:", x and y)  # False (both must be True)
print("OR:", x or y)    # True (at least one must be True)
print("NOT:", not x)    # False (negates the value)

Truth table for logical operators:

  • and: True only if both operands are True
  • or: True if at least one operand is True
  • not: Negates the operand (True becomes False, False becomes True)

Identity Operators

Identity operators check if two variables refer to the same object:

x = [1, 2, 3]
y = [1, 2, 3]
z = x

print("is:", x is z)       # True (same object)
print("is not:", x is y)   # False (different objects, same values)

Membership Operators

Membership operators check if a value is present in a sequence:

fruits = ["apple", "banana", "cherry"]

print("in:", "banana" in fruits)         # True
print("not in:", "orange" not in fruits) # True

Bitwise Operators

Bitwise operators perform operations on binary representations:

a = 5  # Binary: 101
b = 3  # Binary: 011

print("AND:", a & b)   # 1 (001)
print("OR:", a | b)    # 7 (111)
print("XOR:", a ^ b)   # 6 (110)
print("NOT:", ~a)      # -6 (bitwise negation)
print("Left Shift:", a << 1)  # 10 (1010)
print("Right Shift:", a >> 1) # 2 (10)

Operator Precedence

Operators have precedence that determines the order of evaluation:

  1. Parentheses: ()
  2. Exponentiation: **
  3. Unary operators: + - ~
  4. Multiplication, Division, Floor Division, Modulus: * / // %
  5. Addition, Subtraction: + -
  6. Bitwise shifts: << >>
  7. Bitwise AND: &
  8. Bitwise XOR: ^
  9. Bitwise OR: |
  10. Comparison operators: <= < > >=
  11. Equality operators: == !=
  12. Membership operators: in, not in
  13. Identity operators: is, is not
  14. Logical NOT: not
  15. Logical AND: and
  16. Logical OR: or
# Examples of precedence
result1 = 2 + 3 * 4     # 14 (multiplication first)
result2 = (2 + 3) * 4   # 20 (parentheses first)
result3 = 2 ** 3 ** 2   # 512 (right-associative)

Mini Project

Create a simple calculator that demonstrates operator usage:

  • Ask the user for two numbers and an operation
  • Support addition, subtraction, multiplication, division
  • Handle division by zero
  • Display the result with proper formatting
  • Bonus: Add more operations (modulus, exponentiation, floor division)

Here's a starting template:

# Simple calculator
print("Simple Calculator")
print("Operations: +, -, *, /, %, //, **")

num1 = float(input("Enter first number: "))
operation = input("Enter operation: ")
num2 = float(input("Enter second number: "))

if operation == "+":
    result = num1 + num2
    print(f"{num1} + {num2} = {result}")
elif operation == "-":
    result = num1 - num2
    print(f"{num1} - {num2} = {result}")
elif operation == "*":
    result = num1 * num2
    print(f"{num1} * {num2} = {result}")
elif operation == "/":
    if num2 != 0:
        result = num1 / num2
        print(f"{num1} / {num2} = {result}")
    else:
        print("Error: Division by zero")
elif operation == "%":
    result = num1 % num2
    print(f"{num1} % {num2} = {result}")
elif operation == "//":
    result = num1 // num2
    print(f"{num1} // {num2} = {result}")
elif operation == "**":
    result = num1 ** num2
    print(f"{num1} ** {num2} = {result}")
else:
    print("Invalid operation")

Common Mistakes

  • Confusing assignment (=) with comparison (==)
  • Forgetting operator precedence
  • Mixing data types inappropriately
  • Not handling division by zero
  • Using bitwise operators on floating-point numbers
