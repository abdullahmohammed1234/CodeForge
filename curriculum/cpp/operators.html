C++ Operators

Operators in C++ are symbols that perform operations on variables and values. They are essential for performing calculations, making comparisons, and controlling program flow. Understanding operator precedence and associativity is crucial for writing correct expressions.

Arithmetic Operators

Arithmetic operators perform mathematical operations:

Operator Description Example Result
+ Addition 5 + 3 8
- Subtraction 5 - 3 2
* Multiplication 5 * 3 15
/ Division 5 / 3 1 (integer division)
% Modulus (remainder) 5 % 3 2

Arithmetic Operator Examples

#include 

int main() {
    int a = 10, b = 3;
    
    std::cout << "Addition: " << a + b << std::endl;       // 13
    std::cout << "Subtraction: " << a - b << std::endl;    // 7
    std::cout << "Multiplication: " << a * b << std::endl; // 30
    std::cout << "Division: " << a / b << std::endl;       // 3 (integer)
    std::cout << "Modulus: " << a % b << std::endl;       // 1
    
    // Floating-point division
    double x = 10.0, y = 3.0;
    std::cout << "Float division: " << x / y << std::endl; // 3.333...
    
    return 0;
}

Assignment Operators

Assignment operators assign values to variables:

Operator Example Equivalent to
= x = 5 x = 5
+= x += 3 x = x + 3
-= x -= 3 x = x - 3
*= x *= 3 x = x * 3
/= x /= 3 x = x / 3
%= x %= 3 x = x % 3

Assignment Operator Examples

int x = 10;

// Compound assignment
x += 5;  // x = 15
x -= 3;  // x = 12
x *= 2;  // x = 24
x /= 4;  // x = 6
x %= 5;  // x = 1

Comparison (Relational) Operators

Comparison operators compare two values and return true or false:

Operator Description Example Result
== Equal to 5 == 3 false
!= Not equal to 5 != 3 true
< Less than 5 < 3 false
> Greater than 5 > 3 true
<= Less than or equal to 5 <= 5 true
>= Greater than or equal to 5 >= 3 true

Comparison Examples

#include <iostream>

int main() {
    int a = 5, b = 10;
    
    std::cout << std::boolalpha;  // Print bool as true/false
    
    std::cout << "a == b: " << (a == b) << std::endl;  // false
    std::cout << "a != b: " << (a != b) << std::endl;  // true
    std::cout << "a < b: " << (a < b) << std::endl;    // true
    std::cout << "a > b: " << (a > b) << std::endl;    // false
    std::cout << "a <= b: " << (a <= b) << std::endl;  // true
    std::cout << "a >= b: " << (a >= b) << std::endl;  // false
    
    return 0;
}

Logical Operators

Logical operators combine boolean expressions:

Operator Description Example Result
&& Logical AND true && false false
|| Logical OR true || false true
! Logical NOT !true false

Logical Operator Examples

#include <iostream>

int main() {
    bool a = true, b = false;
    
    std::cout << std::boolalpha;
    
    std::cout << "a && b: " << (a && b) << std::endl;  // false
    std::cout << "a || b: " << (a || b) << std::endl;  // true
    std::cout << "!a: " << (!a) << std::endl;        // false
    std::cout << "!b: " << (!b) << std::endl;        // true
    
    // Short-circuit evaluation
    int x = 5;
    bool result = (x > 10) && (x++ > 0);  // x++ not executed
    std::cout << "x after short-circuit: " << x << std::endl;
    
    return 0;
}

Increment and Decrement Operators

Increment (++) and decrement (--) operators modify variables by 1:

int x = 5;

// Post-increment: use value, then increment
int y = x++;  // y = 5, x = 6

// Pre-increment: increment, then use value
int z = ++x;  // z = 7, x = 7

// Post-decrement
y = x--;      // y = 7, x = 6

// Pre-decrement
z = --x;      // z = 5, x = 5

Bitwise Operators

Bitwise operators work on individual bits of integer values:

Operator Description Example
& Bitwise AND 5 & 3 = 1
| Bitwise OR 5 | 3 = 7
^ Bitwise XOR 5 ^ 3 = 6
~ Bitwise NOT ~5 = -6
<< Left shift 5 << 1 = 10
>> Right shift 5 >> 1 = 2

Bitwise Examples

#include <iostream>

int main() {
    int a = 5;  // Binary: 101
    int b = 3;  // Binary: 011
    
    std::cout << "a & b: " << (a & b) << std::endl;   // 1 (001)
    std::cout << "a | b: " << (a | b) << std::endl;   // 7 (111)
    std::cout << "a ^ b: " << (a ^ b) << std::endl;   // 6 (110)
    std::cout << "~a: " << (~a) << std::endl;         // Bitwise NOT
    std::cout << "a << 1: " << (a << 1) << std::endl; // 10 (left shift)
    std::cout << "a >> 1: " << (a >> 1) << std::endl; // 2 (right shift)
    
    return 0;
}

Ternary Conditional Operator

The ternary operator (? :) provides a shorthand for if-else:

// Syntax: condition ? value_if_true : value_if_false

int x = 10, y = 20;
int max = (x > y) ? x : y;  // max = 20

std::string result = (x % 2 == 0) ? "even" : "odd";

// Nested ternary (use with caution)
int grade = 85;
std::string letter = (grade >= 90) ? "A" : 
                    (grade >= 80) ? "B" : 
                    (grade >= 70) ? "C" : "F";

Operator Precedence

Operator precedence determines the order of evaluation in expressions:

  1. Postfix increment/decrement: ++ --
  2. Unary operators: + - ! ~ ++ -- (pre)
  3. Multiplicative: * / %
  4. Additive: + -
  5. Shift: << >>
  6. Relational: < <= > >=
  7. Equality: == !=
  8. Bitwise AND: &
  9. Bitwise XOR: ^
  10. Bitwise OR: |
  11. Logical AND: &&
  12. Logical OR: ||
  13. Ternary: ? :
  14. Assignment: = += -= *= /= %=
  15. Comma: ,

Precedence Examples

// Multiplication before addition
int result = 2 + 3 * 4;  // 14, not 20

// Parentheses override precedence
int result2 = (2 + 3) * 4;  // 20

// Logical operators have lower precedence than comparison
bool result3 = x > 5 && y < 10;  // Correct
// bool result4 = x > 5 && y < 10;  // Same as above

Example Program

#include <iostream>

int main() {
    int a, b;
    std::cout << "Enter two numbers: ";
    std::cin >> a >> b;
    
    // Arithmetic operations
    std::cout << "Sum: " << a + b << std::endl;
    std::cout << "Difference: " << a - b << std::endl;
    std::cout << "Product: " << a * b << std::endl;
    if (b != 0) {
        std::cout << "Division: " << a / b << std::endl;
        std::cout << "Remainder: " << a % b << std::endl;
    }
    
    // Comparisons
    std::cout << std::boolalpha;
    std::cout << a << " > " << b << ": " << (a > b) << std::endl;
    std::cout << a << " == " << b << ": " << (a == b) << std::endl;
    
    // Ternary operator
    std::string comparison = (a > b) ? "greater than" : 
                           (a < b) ? "less than" : "equal to";
    std::cout << a << " is " << comparison << " " << b << std::endl;
    
    return 0;
}

Common Mistakes

  • Confusing = (assignment) with == (comparison)
  • Misunderstanding operator precedence
  • Integer division when float division is needed
  • Using bitwise operators on signed integers carelessly
  • Forgetting parentheses in complex expressions

Best Practices

  • Use parentheses to clarify complex expressions
  • Avoid overly complex expressions in one line
  • Use compound assignment operators when appropriate
  • Be careful with floating-point comparisons
  • Use meaningful variable names to clarify operations
  • Test edge cases (division by zero, overflow, etc.)
