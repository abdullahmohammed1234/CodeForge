C++ Data Types

Data types in C++ specify the type of data that a variable can hold. They determine the size, range, and operations that can be performed on the data. C++ provides fundamental data types and allows creation of user-defined types.

Fundamental Data Types

C++ has several built-in data types that are part of the language:

Integer Types

Type Size Range Description
int 4 bytes -2,147,483,648 to 2,147,483,647 Basic integer type
short 2 bytes -32,768 to 32,767 Short integer
long 4-8 bytes -2,147,483,648 to 2,147,483,647 (or larger) Long integer
long long 8 bytes -9,223,372,036,854,775,808 to 9,223,372,036,854,775,807 Very long integer

Floating-Point Types

Type Size Precision Range
float 4 bytes 6-7 decimal digits ±3.4e±38
double 8 bytes 15 decimal digits ±1.7e±308
long double 8-16 bytes 18-19 decimal digits ±1.1e±4932

Character Types

Type Size Range Description
char 1 byte -128 to 127 or 0 to 255 Single character
wchar_t 2-4 bytes Larger character set Wide character

Boolean Type

Type Size Values Description
bool 1 byte true, false Boolean values

Type Modifiers

Type modifiers can be used with fundamental types to modify their properties:

  • signed: Allows negative values (default for int)
  • unsigned: Only positive values, doubles the range
  • short: Reduces size
  • long: Increases size
// Examples:
signed int x;      // Same as int x
unsigned int y;    // Only positive values
short int z;       // Same as short z
long int w;        // Same as long w
unsigned long long big_num;

Void Type

The void type represents the absence of type. It's used for:

  • Functions that don't return a value
  • Generic pointers
  • Function parameters (rare)
void myFunction() {
    // Function that doesn't return anything
}

void* generic_ptr;  // Can point to any type

Sizeof Operator

The sizeof operator returns the size of a data type or variable in bytes:

#include <iostream>

int main() {
    std::cout << "Size of int: " << sizeof(int) << " bytes" << std::endl;
    std::cout << "Size of double: " << sizeof(double) << " bytes" << std::endl;
    std::cout << "Size of char: " << sizeof(char) << " bytes" << std::endl;
    
    int x;
    std::cout << "Size of variable x: " << sizeof(x) << " bytes" << std::endl;
    
    return 0;
}

Type Aliases

You can create aliases for existing types using typedef or using:

// Using typedef
typedef unsigned long ulong;
typedef int integer;

// Using 'using' (C++11)
using ullong = unsigned long long;
using byte = unsigned char;

// Usage
ulong big_number;
integer count;
ullong very_big;
byte data;

Auto Type Deduction

C++11 introduced auto keyword for automatic type deduction:

auto x = 5;        // int
auto y = 3.14;     // double
auto z = 'A';      // char
auto flag = true;  // bool

// Useful with complex types
std::vector<int> numbers = {1, 2, 3, 4, 5};
auto it = numbers.begin();  // Iterator type

User-Defined Types

C++ allows creation of custom data types:

  • Structures (struct): Group related data
  • Classes: Object-oriented programming
  • Unions: Share memory between variables
  • Enumerations (enum): Named constants

Structures

struct Person {
    std::string name;
    int age;
    double height;
};

// Usage
Person john;
john.name = "John";
john.age = 30;
john.height = 5.9;

Enumerations

enum Color { RED, GREEN, BLUE };
enum Status { ACTIVE = 1, INACTIVE = 0, PENDING = 2 };

// Usage
Color my_color = RED;
Status user_status = ACTIVE;

Type Conversion

C++ supports implicit and explicit type conversion:

Implicit Conversion

int x = 5;
double y = x;  // int to double (safe)

char c = 'A';
int ascii = c;  // char to int (ASCII value)

Explicit Conversion (Casting)

double pi = 3.14159;
int approx = (int)pi;  // C-style cast

// C++ style casts
int value = static_cast<int>(pi);
int* ptr = reinterpret_cast<int*>(some_void_ptr);

Example Program

#include <iostream>
#include <iomanip>  // For setprecision

int main() {
    // Different data types
    int integer_num = 42;
    double float_num = 3.14159;
    char character = 'C';
    bool boolean = true;
    std::string text = "Hello";
    
    // Display sizes
    std::cout << "Sizes in bytes:" << std::endl;
    std::cout << "int: " << sizeof(int) << std::endl;
    std::cout << "double: " << sizeof(double) << std::endl;
    std::cout << "char: " << sizeof(char) << std::endl;
    std::cout << "bool: " << sizeof(bool) << std::endl;
    
    // Display values
    std::cout << "\nValues:" << std::endl;
    std::cout << "Integer: " << integer_num << std::endl;
    std::cout << "Float: " << std::fixed << std::setprecision(2) << float_num << std::endl;
    std::cout << "Character: " << character << std::endl;
    std::cout << "Boolean: " << (boolean ? "true" : "false") << std::endl;
    std::cout << "String: " << text << std::endl;
    
    return 0;
}

Common Mistakes

  • Overflow: Assigning values outside type range
  • Precision loss: Converting float to int
  • Uninitialized variables: May contain garbage values
  • Type mismatches in operations
  • Wrong size assumptions: Sizes vary by platform

Best Practices

  • Choose appropriate data types for your needs
  • Use unsigned types when negative values aren't needed
  • Initialize variables to avoid undefined behavior
  • Use meaningful type aliases for complex types
  • Consider memory usage in resource-constrained environments
