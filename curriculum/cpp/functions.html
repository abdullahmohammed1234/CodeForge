C++ Functions

Functions are the building blocks of C++ programs. They allow you to organize code into reusable, modular units that perform specific tasks. Understanding functions is essential for writing maintainable and efficient C++ code.

What is a Function?

A function is a named block of code that performs a specific task. Functions help:

  • Modularity: Break programs into manageable pieces
  • Reusability: Use the same code multiple times
  • Readability: Make code easier to understand
  • Testing: Test individual components
  • Maintenance: Easier to modify and debug

Function Declaration and Definition

Functions in C++ are declared and defined separately:

Function Declaration (Prototype)

// Declaration (usually in header files)
return_type function_name(parameter_list);

// Examples:
int add(int a, int b);
void printMessage(std::string message);
double calculateArea(double radius);

Function Definition

// Definition (implementation)
return_type function_name(parameter_list) {
    // Function body
    // Statements...
    return value;  // If return type is not void
}

// Example:
int add(int a, int b) {
    int sum = a + b;
    return sum;
}

Parameters and Arguments

Parameters are variables declared in the function definition. Arguments are values passed to the function:

Pass by Value

void increment(int x) {
    x++;  // Changes local copy only
    std::cout << "Inside function: " << x << std::endl;
}

int main() {
    int num = 5;
    increment(num);  // Pass copy of num
    std::cout << "After function: " << num << std::endl;  // Still 5
    return 0;
}

Pass by Reference

void increment(int& x) {
    x++;  // Changes original variable
    std::cout << "Inside function: " << x << std::endl;
}

int main() {
    int num = 5;
    increment(num);  // Pass reference to num
    std::cout << "After function: " << num << std::endl;  // Now 6
    return 0;
}

Pass by Pointer

void increment(int* ptr) {
    (*ptr)++;  // Dereference and increment
}

int main() {
    int num = 5;
    increment(&num);  // Pass address of num
    std::cout << "After function: " << num << std::endl;  // Now 6
    return 0;
}

Return Values

Functions can return values using the return statement:

// Function returning int
int square(int x) {
    return x * x;
}

// Function returning void (no return value)
void printHello() {
    std::cout << "Hello, World!" << std::endl;
    // No return statement needed for void functions
}

// Multiple return statements
bool isEven(int x) {
    if (x % 2 == 0) {
        return true;
    } else {
        return false;
    }
}

Function Overloading

Function overloading allows multiple functions with the same name but different parameters:

// Overloaded functions
int add(int a, int b) {
    return a + b;
}

double add(double a, double b) {
    return a + b;
}

int add(int a, int b, int c) {
    return a + b + c;
}

int main() {
    std::cout << add(2, 3) << std::endl;        // Calls int version
    std::cout << add(2.5, 3.7) << std::endl;    // Calls double version
    std::cout << add(1, 2, 3) << std::endl;     // Calls three-parameter version
    return 0;
}

Default Parameters

Functions can have default values for parameters:

void printMessage(std::string message, int count = 1) {
    for (int i = 0; i < count; i++) {
        std::cout << message << std::endl;
    }
}

int main() {
    printMessage("Hello");        // Uses default count = 1
    printMessage("Hi", 3);        // Overrides default
    return 0;
}

Function Templates

Templates allow functions to work with different data types:

// Function template
template <typename T>
T maximum(T a, T b) {
    return (a > b) ? a : b;
}

// Template with multiple parameters
template <typename T, typename U>
void printPair(T first, U second) {
    std::cout << first << ", " << second << std::endl;
}

int main() {
    std::cout << maximum(5, 3) << std::endl;           // int
    std::cout << maximum(5.5, 3.2) << std::endl;       // double
    std::cout << maximum('z', 'a') << std::endl;        // char
    
    printPair(42, "hello");        // int and string
    printPair(3.14, 'x');          // double and char
    
    return 0;
}

Recursion

Recursive functions call themselves:

// Factorial using recursion
int factorial(int n) {
    if (n <= 1) {
        return 1;  // Base case
    } else {
        return n * factorial(n - 1);  // Recursive call
    }
}

// Fibonacci sequence
int fibonacci(int n) {
    if (n <= 1) {
        return n;
    } else {
        return fibonacci(n - 1) + fibonacci(n - 2);
    }
}

int main() {
    std::cout << "Factorial of 5: " << factorial(5) << std::endl;
    std::cout << "Fibonacci 8: " << fibonacci(8) << std::endl;
    return 0;
}

Lambda Functions (C++11)

Lambda functions are anonymous functions defined inline:

#include <algorithm>
#include <vector>

int main() {
    std::vector<int> numbers = {3, 1, 4, 1, 5, 9, 2, 6};
    
    // Lambda for sorting in descending order
    std::sort(numbers.begin(), numbers.end(), 
              [](int a, int b) { return a > b; });
    
    // Lambda with capture
    int threshold = 5;
    auto count = std::count_if(numbers.begin(), numbers.end(),
                              [threshold](int x) { return x > threshold; });
    
    std::cout << "Numbers greater than " << threshold << ": " << count << std::endl;
    
    return 0;
}

Function Pointers

Function pointers store addresses of functions:

// Function pointer declaration
int (*funcPtr)(int, int);

// Assign function address
funcPtr = &add;  // or just funcPtr = add

// Call through pointer
int result = funcPtr(5, 3);

// Function that takes function pointer
void processNumbers(int a, int b, int (*operation)(int, int)) {
    int result = operation(a, b);
    std::cout << "Result: " << result << std::endl;
}

// Usage
processNumbers(10, 5, add);        // Addition
processNumbers(10, 5, subtract);   // Subtraction

Inline Functions

Inline functions suggest to the compiler to insert code directly:

// Inline function
inline int square(int x) {
    return x * x;
}

// Benefits: Faster execution for small functions
// Drawbacks: Code bloat if overused

Function Best Practices

  • Single Responsibility: Each function should do one thing
  • Descriptive Names: Use clear, meaningful function names
  • Proper Parameters: Use const references for large objects
  • Documentation: Comment complex functions
  • Avoid Global Variables: Pass data through parameters
  • Limit Parameters: Keep parameter count reasonable

Example Program

#include <iostream>
#include <string>
#include <vector>

// Function declarations
double calculateAverage(const std::vector<int>& numbers);
void printResults(const std::string& label, double value);
int findMaximum(const std::vector<int>& numbers);

// Function definitions
double calculateAverage(const std::vector<int>& numbers) {
    if (numbers.empty()) return 0.0;
    
    int sum = 0;
    for (int num : numbers) {
        sum += num;
    }
    return static_cast<double>(sum) / numbers.size();
}

void printResults(const std::string& label, double value) {
    std::cout << label << ": " << value << std::endl;
}

int findMaximum(const std::vector<int>& numbers) {
    if (numbers.empty()) return 0;
    
    int max = numbers[0];
    for (size_t i = 1; i < numbers.size(); i++) {
        if (numbers[i] > max) {
            max = numbers[i];
        }
    }
    return max;
}

int main() {
    std::vector<int> scores = {85, 92, 78, 96, 88, 73, 91};
    
    double average = calculateAverage(scores);
    int maximum = findMaximum(scores);
    
    printResults("Average Score", average);
    printResults("Highest Score", maximum);
    
    return 0;
}
