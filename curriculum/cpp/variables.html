C++ Variables

Variables in C++ are named storage locations that hold data values. They allow you to store, retrieve, and manipulate data throughout your program. Understanding variables is fundamental to programming in C++.

What is a Variable?

A variable is a container for storing data values. In C++, you must declare a variable before you can use it, specifying its data type and name. Variables can hold different types of data such as numbers, text, or boolean values.

Variable Declaration

In C++, variables must be declared before use. The declaration tells the compiler the variable's name and data type.

// Variable declaration syntax:
// data_type variable_name;

// Examples:
int age;
double salary;
char grade;
std::string name;

Variable Initialization

You can initialize a variable at the time of declaration or later in the program.

// Declaration and initialization together:
int age = 25;
double pi = 3.14159;
char letter = 'A';
std::string greeting = "Hello";

// Declaration first, initialization later:
int score;
score = 100;

Variable Naming Rules

  • Variable names can contain letters, digits, and underscores
  • Variable names must begin with a letter or underscore
  • Variable names are case-sensitive (age and Age are different)
  • Cannot use C++ keywords as variable names
  • Use meaningful names that describe the variable's purpose
// Valid variable names:
int user_age;
double _temperature;
std::string firstName;
char grade_A;

// Invalid variable names:
// int 123abc;     // Starts with digit
// int my-age;     // Contains hyphen
// int int;        // Reserved keyword

Data Types

C++ has several built-in data types for different kinds of data:

  • int: Integer numbers (whole numbers)
  • double/float: Floating-point numbers (decimals)
  • char: Single characters
  • bool: Boolean values (true/false)
  • std::string: Text strings

Variable Scope

The scope of a variable determines where it can be accessed in the program:

  • Local variables: Declared inside functions, accessible only within that function
  • Global variables: Declared outside functions, accessible throughout the program
  • Block scope: Variables declared inside blocks (like loops or if statements)
#include <iostream>

int global_var = 10; // Global variable

int main() {
    int local_var = 20; // Local variable
    
    if (true) {
        int block_var = 30; // Block scope variable
        std::cout << block_var << std::endl;
    }
    
    // block_var is not accessible here
    std::cout << local_var << std::endl;
    std::cout << global_var << std::endl;
    
    return 0;
}

Constants

Constants are variables whose values cannot be changed after initialization. Use the const keyword:

const double PI = 3.14159;
const int MAX_USERS = 100;
const std::string COMPANY_NAME = "CodeForge";

// PI = 3.14; // Error: cannot modify constant

Example Program

#include <iostream>
#include <string>

int main() {
    // Variable declarations
    std::string name;
    int age;
    double height;
    char grade;
    bool is_student;
    
    // Variable assignments
    name = "Alice";
    age = 20;
    height = 5.6;
    grade = 'A';
    is_student = true;
    
    // Display variables
    std::cout << "Name: " << name << std::endl;
    std::cout << "Age: " << age << std::endl;
    std::cout << "Height: " << height << " feet" << std::endl;
    std::cout << "Grade: " << grade << std::endl;
    std::cout << "Is student: " << (is_student ? "Yes" : "No") << std::endl;
    
    return 0;
}

Common Mistakes

  • Using variables before declaring them
  • Mixing up data types (assigning string to int)
  • Forgetting to initialize variables (may contain garbage values)
  • Using reserved keywords as variable names
  • Scope issues (trying to access variables outside their scope)
